Chapitre 2: Géométrie plane

1. Introduction aux figures géométriques de base

La géométrie plane étudie les formes et les figures dans un espace à deux dimensions. Commençons par explorer les figures de base :

  • Le point
  • La droite
  • Le segment
  • Le cercle
  • Le triangle
  • Le carré
  • Le rectangle

2. Le point et la droite

Un point est la figure géométrique la plus simple. Il n'a pas de dimension, seulement une position.

Une droite est une ligne qui s'étend à l'infini dans les deux directions. Elle n'a pas de début ni de fin.

Dans cette figure, le point rouge A est situé sur la droite.

3. Le segment

Un segment est une partie d'une droite comprise entre deux points, appelés extrémités du segment.

Le segment [BC] est représenté ci-dessus.

4. Le cercle

Un cercle est l'ensemble des points situés à égale distance d'un point fixe appelé centre. Cette distance est appelée rayon.

O est le centre du cercle et r représente le rayon.
