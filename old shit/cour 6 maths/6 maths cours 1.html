Chapitre 1: Les nombres entiers et décimaux

1. Introduction aux nombres entiers

Les nombres entiers sont des nombres sans partie décimale. Ils peuvent être positifs, négatifs ou zéro.

Exemples de nombres entiers : -3, -2, -1, 0, 1, 2, 3, 4, 5, ...

2. Opérations sur les nombres entiers

Nous pouvons effectuer différentes opérations avec les nombres entiers :

  • Addition (+)
  • Soustraction (-)
  • Multiplication (×)
  • Division (/)

5 + 3 = 8

7 - 4 = 3

6 × 2 = 12

8 ÷ 2 = 4

3. Les nombres décimaux

Les nombres décimaux sont des nombres qui ont une partie entière et une partie décimale, séparées par une virgule.

Exemples de nombres décimaux : 3,14 ; 0,5 ; 2,75 ; 10,01

4. Opérations sur les nombres décimaux

Nous pouvons effectuer les mêmes opérations avec les nombres décimaux qu'avec les nombres entiers.

2,5 + 1,7 = 4,2

5,8 - 3,2 = 2,6

1,5 × 2 = 3,0

4,8 ÷ 2 = 2,4
